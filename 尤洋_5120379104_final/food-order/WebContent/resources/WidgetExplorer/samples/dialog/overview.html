This sample illustrates the default wijdialog widget, which is useful for displaying information such as error messages or instructions. The dialog window can be moved, resized and closed with the 'x' icon. Other samples in the wijdialog section will demonstrate how to use animation effects, modal dialog boxes, and iFrame elements.
