Overview

This sample demonstrates how easyly you can create a cross-browser Line Chart using Wijmo. This Line Chart that shows cars production worldwide through the years.

Test the features

  • Hover over a data point to see the tooltip.
  • Check the box under the chart and apply changes to compare default and spline Line Chart.
  • Note the formatted Dates on the x axis. The original values are Date Obects. This is done automatically by the chart to optimize display. It can also be set.
