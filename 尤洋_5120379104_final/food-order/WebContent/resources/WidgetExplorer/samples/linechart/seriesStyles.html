This sample demonstrates how easy you can customize lines visualization using Wijmo Line Chart and make every graph look unique.

Test the features

  • Note the colors of each line in chart area and in the legend.
  • Note lines highlighting on mouse hover - they become thicker and clear.
  • Play with controls below to see all the divercity of display options Wijmo Line Chart offer.
  • Note how lines become wider when you change 'stroke-width' option
  • Note you can change line paleness and brightness using 'stroke-opacity' option.
  • Note how easy you can decorate lines using 'stroke-dasharray' option.
  • Note the legend elements updating according to selected 'stroke-dasharray' option.
  • Note you can use same properties to setup lines visualization on mouse hover.
  • Note you can use same stroke proprties to setup visualization of any kind Wijmo Line Chart lines including Grid And Ticks
  • Go to 'Sources' tab to see more instructions
