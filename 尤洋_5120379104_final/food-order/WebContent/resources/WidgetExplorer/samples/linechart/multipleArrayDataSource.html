Shared DataSource

Besides using of shared array Wijmo Line Chart API also allows you to use multiple arrays as data source. Just define several arrays and bind the data easyly using the same 'dataSource' property, but this time inside the 'seriesList' property.

Test the features

  • Note both 'x' and 'y' axises are presented as continuous number variables.
  • Go to the 'Sources' tab to see how to setup chart data using multiple arrays as data source.
