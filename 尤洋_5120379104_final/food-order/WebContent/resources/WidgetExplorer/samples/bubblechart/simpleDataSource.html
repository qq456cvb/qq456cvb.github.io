Wijmo BubbleChart is the perfect tool for data visualization. In this sample series you will learn how to setup any chart base - the data you need to present. Wijmo Bubble Chart API support several ways to setup data source. The first of them is using of 'seriesList' property.

Test the features

  • Note all chart bubbles are grouped in series with different colors.
  • Note the 'x' axis is presented as determined string values and the 'y' axis contains numbers and is continuous .
  • Note every chart bubble has it own size according to the third data value.
  • Hover over a bubble to see the tooltip with description.
  • Go to the 'Sources' tab to see how to setup chart data using 'seriesList' property.
