Multiple Array Datasource

Besides using of shared array Wijmo Bubble Chart API also allows you to use multiple arrays as data source. Just define several arrays and bind the data easyly using the same 'dataSource' property, but this time inside the 'seriesList' property.

Test the features

  • Note all chart bubbles are grouped in series with different colors.
  • Note the 'x' axis is presented as determined string values and the 'y' axis contains numbers and is continuous .
  • Note every chart bubble has it own size according to the third data value.
  • Hover over a bubble to see the tooltip with description.
  • Go to the 'Sources' tab to see how to setup chart data using multiple arrays as data source.
