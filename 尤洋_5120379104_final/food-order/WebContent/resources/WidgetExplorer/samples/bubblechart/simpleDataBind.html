Array Datasource

This sample shows another way how to setup chart data. Besides defining data directly in 'seriesList' property you can use array as data source and data binding. All you need is to use 'dataSource' property.

Test the features

  • Note both 'x' and 'y' axises are presented as continuous number variables.
  • Note every chart bubble has it own size according to the third data value.
  • Hover over a bubble to see the tooltip with description.
  • Go to the 'Sources' tab to see how to bind array data using 'dataSource' property.
