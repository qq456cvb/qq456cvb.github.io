Remote data using OData API

The sample shows how to load data from the OData service using wijmo.data.ODataView. To activate paging and sorting it is sufficient to activate the allowPaging and allowSorting options.

To use ODataView or AjaxDataView you need to add the wijmo.data.ajax.js dependency. This is not included in our combined file and must be added separately. Read our Data Quick Start to see the references to use.
