Data Types

In this sample, the columns data types in the wijgrid are explicitly specified:

  • ProductName: dataType = "string"
  • UnitPrice: dataType = "currency"
  • Quantity: dataType = "number", dataFormatString = "n0"
  • Discount: dataType = "number", dataFormatString = "p0"
  • OrderDate: dataType = "datetime", dataFormatString = "MMM-dd-yyyy"
  • Overseas: dataType = "boolean"
